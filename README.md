# 🧠 NLP-Based Automated FIR Generation System

## 📌 Overview
This project is an AI-powered system that converts natural language complaints into structured First Information Reports (FIRs). It uses Natural Language Processing (NLP) and Machine Learning techniques to extract key information and generate legally formatted documents.

The system supports text input, live voice recording, and audio file upload, making it accessible and user-friendly.

---

## 🚀 Features
- Text-based complaint input  
- Live voice recording  
- Audio file upload support  
- NLP-based entity extraction (NER)  
- Automated FIR document generation  
- Multilingual support  
- Secure data storage  

---

## 🛠️ Technologies Used
- Python  
- spaCy, NLTK, Transformers  
- BERT / Legal-BERT  
- Streamlit  
- SQLite
